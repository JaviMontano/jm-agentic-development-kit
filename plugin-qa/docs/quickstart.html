Quickstart

Create, validate, audit, and fix a plugin in 5 steps

Create a Plugin from Scratch

Describe what you want in natural language. PQA runs the full lifecycle: ideation, planning, design, and scaffold generation.

claude — /pqa:create
$ /pqa:create "A code review plugin that checks for security patterns" IDEATE Generating plugin concept... ✓ Plugin name: security-review ✓ 3 agents defined ✓ 8 skills mapped PLAN Architecture planning... ✓ MOAT strategy: L2 depth ✓ Command tree: 12 commands BUILD Scaffolding plugin... ✓ Created security-review/.claude-plugin/plugin.json ✓ Created 3 agent definitions ✓ Created 8 skill MOAT directories ✓ Created hooks/hooks.json ✓ Created CLAUDE.md DONE Plugin scaffold ready at ./security-review/
Tip: Use /pqa:lifecycle for the full 9-movement pipeline with quality gates.

Validate Structure

Run structural validation to ensure your plugin conforms to the Claude Code plugin specification.

claude — /pqa:validate
$ /pqa:validate VALIDATE Running 5 structural checks... ✓ plugin.json — valid schema ✓ agents/ — 3 agents, all have AGENT.md ✓ commands/ — 12 commands, all linked ✓ skills/ — 8 skills, MOAT structure OK ⚠ hooks/ — missing PreCompact handler Result: 4 passed · 1 warning · 0 errors

Full Audit

Run the 20-skill quality audit covering structure, content, conformance, hooks, and cross-references.

claude — /pqa:audit
$ /pqa:audit AUDIT Running 20-skill quality audit... Category Score Structure 92/100 Content Quality 87/100 Conformance 95/100 Hook Coverage 78/100 Cross-References 90/100 Overall: 88/100 — Grade: A- 3 issues found. Run /pqa:fix to auto-repair.

Auto-Fix Issues

Let PQA detect and repair common plugin issues automatically.

claude — /pqa:fix
$ /pqa:fix FIX Scanning for auto-fixable issues... ✓ Added PreCompact hook to hooks.json ✓ Created examples/ in skills/pattern-scan/ ✓ Created examples/ in skills/vuln-check/ ✓ Linked /sr:debug to agent debug-agent Fixed: 4 issues · Remaining: 0 New score: 96/100 — Grade: A+
