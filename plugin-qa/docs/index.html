4 Specialized Agents

Each agent owns a segment of the plugin lifecycle

◎

Plugin Architect

Designs plugin structure, plans MOAT strategy, defines agent and skill topology. Owns IDEATE, PLAN, and DESIGN movements.

★

QA Engineer

Validates structure, audits content quality, generates scorecards, and runs conformance checks. Owns VALIDATE, AUDIT, and REPORT.

✎

Spec Writer

Authors agent commands, skill MOAT specifications, hook configurations, and manifest files. Owns SPECIFY movement.

⚙

Builder Agent

Scaffolds plugin directories, generates MOAT assets, installs hooks, and applies auto-fixes. Owns BUILD and FIX movements.
