Key Features

Per-Prompt Heartbeat

Ambient intelligence runs on every prompt. Detects stale artifacts, missing files, health regression. Zero LLM cost.

Knowledge Graph

Traces Principles to Requirements to Tests to Tasks. Detects orphans automatically. Force-directed SVG visualization.

Command Center

Multi-HTML micro-frontend dashboard with 11 views: pipeline, specs, quality, intelligence, workspace, governance.

RAG Memory

Session inputs captured as structured memory files with abstracts, key takeaways, and full content. Cross-session knowledge.
